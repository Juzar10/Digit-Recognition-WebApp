Digit Recognition WebApp

Using Tensorflow and Flask

How to Use it?

  • You can draw an digit on the box above Using mouse or touchpad and then press button called execute to see the result.

  • Press button Clear to clear the box.

    • How it Works?

  • It generates the Image through canvas element of HTML and then it sends that Image to flask file.

  • Then flask files does computations on image using library like Pillow and Tensorflow.

  • And Result is returned Using ajax functionality of jquery library
